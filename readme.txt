
-------------------------- generate the  incomplete multiview dataset --------------------------- 
1. Open folder "generate_dataset". 
2. Download the dataset from the Internet.  (Cornell {http://lig-membres.imag.fr/grimal/data.html}, Washington {http://lig-membres.imag.fr/grimal/data.html}, 3 Sources {http://erdos.ucd.ie/datasets/3sources.html}, BBCSport {http://erdos.ucd.ie/datasets/bbc.html}, and Caltech7 {https://drive.google.com/drive/folders/1O_3YmthAZGiq1ZPSdE74R7Nwos2PmnHH})
3. Process these data to get an incomplete multiview data set. We give the processing method on the BBCSport and Washington data sets.

------------------------------------------ run demo --------------------------------------------- 
1. Open folder "run_code/IMNGA". 
2. Run "demo_MNIGA_washington_missing.m"

-------------cite this--------------------
S. Sun and N. Zhang, "Incomplete Multiview Nonnegative Representation Learning With Graph Completion and Adaptive Neighbors," in IEEE Transactions on Neural Networks and Learning Systems, 2022, doi: 10.1109/TNNLS.2022.3201562.
